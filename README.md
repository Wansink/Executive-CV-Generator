# Executive CV Generator

A practical workflow for generating tailored executive-level CVs using structured master resumes, STAR-based achievements, and a guided prompt approach.

## Overview

This repository provides a simple but powerful system to quickly generate a customised executive CV for specific roles. It combines:

- A **master CV** (your complete career history)
- A **STAR-based achievements file** (structured impact examples)
- A **prompt template** for generating a tailored CV using AI tools

## Files in This Repository

- **CV_Master**  
  Your comprehensive career record. Includes full job history, responsibilities, and background.

- **CV_Star**  
  A structured collection of career highlights using the STAR format:
  - Situation  
  - Task  
  - Action  
  - Result  

- **CV_Generator**  
  A prompt template used to generate a targeted CV suited to a specific role.

## How to Use

### Step 1: Download the Files

Download the following three files from this repository:

- CV_Master
- CV_Star
- CV_Generator

### Step 2: Update Your Source Files

#### Update CV_Master
Ensure this file contains:
- Your latest roles
- Key responsibilities
- Skills and experience
- Career chronology

#### Update CV_Star
Add or refine your strongest achievements:
- Focus on measurable outcomes (e.g., revenue growth, cost savings, transformation impact)
- Keep examples relevant to executive-level responsibilities
- Prioritise high-impact, strategic contributions

### Step 3: Prepare for CV Generation

1. Open the CV_Generator file  
2. Copy its full contents into your preferred AI tool (e.g., ChatGPT prompt box)

### Step 4: Add the Role Description (PD)

Take the Position Description (PD) or job ad for the role you are applying for and paste it into the prompt where indicated.

### Step 5: Generate Your Tailored CV

Run the prompt.

The output will be:
- A targeted executive CV
- Aligned to the role requirements
- Highlighting relevant achievements from CV_Star
- Structured based on your CV_Master

## Best Practices

- Keep your CV_Master comprehensive but structured  
- Use metrics wherever possible in CV_Star (e.g., %, $, scale)  
- Tailor the PD input carefully—this drives relevance  
- Review and lightly edit the generated CV for tone and nuance  

## Tips for Strong Results

- Use clear, outcome-focused language in STAR examples  
- Regularly update both Master and Star files  
- Avoid overloading with too many achievements—prioritise impact  
- Ensure alignment between your executive narrative and target role  

## Example Workflow

1. Update your career history in CV_Master  
2. Add 5–10 strong, quantified achievements in CV_Star  
3. Copy CV_Generator into an AI prompt  
4. Paste the job description  
5. Generate and refine your CV  

## Purpose

This system is designed to:
- Save time tailoring CVs
- Improve alignment with executive roles
- Highlight measurable impact
- Maintain consistency across applications
